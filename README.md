# 📦 Smart Logistics & Order Routing System

A robust, pure C++ application designed to simulate modern supply chain and logistics operations. This project showcases the practical application of Core Data Structures implemented from scratch without relying on the Standard Template Library (STL).

## 🛠️ Data Structures Implemented
* **Queue (FIFO):** Handles incoming cargo management (Order Enqueue/Dequeue).
* **Stack (LIFO):** Simulates truck loading and unloading sequences.
* **Singly Linked List:** Manages and prints dynamic delivery routes/stations.
* **Binary Search Tree (BST):** Empowers fast order tracking and lookup with $O(\log n)$ efficiency.

## 🚀 Key Features
* Custom-built OOP structure with separate models for `Order`, `Node`, and custom DS containers.
* Dynamic multi-order generation.
* Memory management using pointers and manual node deletions.
